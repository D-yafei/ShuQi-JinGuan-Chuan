
# 蜀漆·锦官传——非遗漆器共创

非遗漆器共创平台，传承千年漆艺之美，创新现代生活方式。

## 项目简介

本项目致力于传承和推广四川漆器非物质文化遗产，通过数字化平台展示漆器工艺的历史、文化和艺术价值。

## 主要功能

- **故事引力场**：探索漆器背后的历史故事和文化内涵
- **漆百科**：学习漆器工艺知识
- **漆器培训站**：了解漆器培训课程
- **共创回音壁**：参与漆艺共创活动

## 技术栈

- HTML5
- CSS3
- JavaScript

## 访问方式

直接打开 `蜀漆·锦官传——非遗漆器共创.html` 文件即可查看网站。

## 文化传承

四川漆器工艺历史悠久，本项目旨在通过现代技术手段，让更多人了解和热爱这项非遗技艺。
